Apple Vision Pro Engagement Analysis

Overview

This project uses the K-Means clustering algorithm to analyze engagement metrics on tweets related to the Apple Vision Pro launch. The goal was to uncover patterns in how users interacted with promotional content and discussions surrounding the product.

Data & Preprocessing

The analysis utilized a Kaggle dataset containing Twitter engagement metrics (likes, retweets, replies, quotes, views, bookmarks) for tweets mentioning "Apple Vision Pro".

  • Handling missing values through imputation or removal based on metric importance.
  • Scaling metrics using Min-Max normalization to ensure equal weighting in the clustering algorithm.
  • Selecting key metrics relevant to direct user engagement (e.g., Likes, Retweets, Replies, Views).
Methodology & Visualization

The K-Means algorithm was applied to group tweets into 3 distinct clusters based on their engagement profiles. The optimal number of clusters was determined using the Elbow method.

Clusters were visualized using 2D scatter plots generated with Matplotlib and Seaborn:

  • Likes vs. Retweets
  • Replies vs. Quotes
  • Views vs. Bookmarks
Conclusion & Impact

The clustering results effectively segmented tweets into distinct engagement categories (e.g., high-reach/low-interaction, high-discussion, balanced engagement). These insights highlight how influential users and specific content types shape public discourse and can inform future marketing strategies.

Technologies Used

Python Pandas Scikit-learn Matplotlib Seaborn

Music Classifier & Reccomendor

Overview

This project analyzes audio features from uploaded .wav files using Librosa and classifies the music genre via a pre-trained Keras neural network. It serves as the analysis core for a potential music recommendation system.

How It Works

The system processes standard .wav audio files:

  • Loads audio data using Librosa.
  • Extracts key audio features, primarily Mel-Frequency Cepstral Coefficients (MFCCs), which represent the short-term power spectrum of sound.
  • Feeds these features into a pre-trained Convolutional Neural Network (CNN) model built with Keras/TensorFlow.
  • The model predicts the most likely music genre.
Project Role

This component acts as the front-end analysis tool within a larger pipeline:

  • Users can upload a song (.wav).
  • Receives a genre prediction and potentially visual feedback (like the waveform/spectrogram shown).
  • Connects to related notebooks for model training (`music_classifier.ipynb`) and data preprocessing (`music_preprocess.ipynb`).
  • Provides the basis for recommending similar music.
Purpose & Potential

The primary goal is accurate music genre classification from raw audio. This enables applications like automated music library organization, content-based recommendation engines, and music information retrieval systems.

Technologies Used

Python Librosa NumPy TensorFlow/Keras Matplotlib JSON OS

FORSAKEN - Souls-like Game Development

Early Build of the Game - April 7th, 2025

Overview

Forsaken is a souls-like game where the player navigates challenging dungeons, defeating bosses that represent the protagonist's internal conflicts from their past life. Built with Unity, the game explores themes of choice, grief, and redemption, offering unique gameplay mechanics and player agency in the final outcome.

Key Mechanics

Gameplay innovation centers around high-risk, high-reward combat and resource management:

  • Revive slain bosses/enemies as temporary allies.
  • Steal enemy powers upon defeating them.
  • Transform into enemies for a brief period (one-time use per transformation).
  • Unique weapons with stats dependent on the player's current HP/Mana levels.
  • Killing enemies restores HP/Mana, encouraging aggressive play.
  • High-cost spells with cooldowns to ensure strategic use.
Narrative & Themes

The player character, suffering memory loss after death, is offered a path to resurrection by two opposing gods (Life & Death) in exchange for service. As the player clears levels representing the five stages of grief (Denial, Anger, Bargaining, Depression, Acceptance), conversations with a shadowy figure reveal fragments of their past life and the true nature of the gods' divine wager.

The core theme explores the illusion of choice, challenging the player to potentially break free from the gods' predetermined paths and forge their own destiny.

Development & Role

This game was developed as a team project using the Unity engine. My specific responsibilities included:

  • Character Design
  • Core Gameplay Mechanics Implementation
  • Animation Integration
  • Non-Player Character (NPC) Logic
  • World Building Elements
